## ğŸ› Troubleshooting

### App won't start?
```bash
# Clear and reinstall
rm -rf node_modules package-lock.json
npm install
npm start
```

### Port 3000 in use?
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Mac/Linux
lsof -ti:3000 | xargs kill -9
```

### Styles not loading?
- Hard refresh: Ctrl+Shift+R
- Clear browser cache
- Restart dev server

### No assignments generated?
- Check staff availability
- Verify skills match
- Ensure hours available
- Check vendor requirements
